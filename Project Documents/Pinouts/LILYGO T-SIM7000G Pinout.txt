Reserved pins:
Function		GPIO Pin	Description
SD Card MISO		2	Data input for SD card via SPI
SD Card MOSI		15	Data output for SD card via SPI
SD Card SCK		14	Clock signal for SD card via SPI
SD Card CS		13	Chip select for SD card via SPI
SIM7000G RX		26	Receive line for UART communication
SIM7000G TX		27	Transmit line for UART communication
SIM7000G PWR		4	Power control pin for the module
SIM7000G RESET		5	Reset pin for the module
SIM7000G DTR		25	Data Terminal Ready for serial control
On-board LED (Blue)	12	Indicator LED on the board
Battery ADC		35	Analog input for battery voltage sensing
Solar ADC		36	Analog input for solar panel voltage


Modules attached to LILYGO T-SIM7000G V1.1 (ignoring power):
- LED Strip: "WS2812B Data" to "GPIO14"
- ESP32 Device: "RX" to "GPIO32", "TX" to "GPIO33" - (TO BE REPLACED WITH WAVESHARE DISPLAY)
- Adafruit Stemma Speaker: "Signal" to "GPIO25"
- BNO086 Compass: "SCL" to "GPIO22", "SDA" to "GPIO21", "INT" to "GPIO19", "RST" to "GPIO23"
(Keep in mind that BNO086 RST should be connected to a GPIO pin and NOT the RST pin on the LILYGO ESP32!)
